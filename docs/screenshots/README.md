# Credora release screenshots

This directory is the deliberate home for portfolio and release imagery. Capture these routes from the Docker-backed app before a public release:

- `/` — cinematic landing and simulator boundary.
- `/demo` and `/analyzer` — guest handoff and educational risk workflow.
- `/results/<analysisId>` — risk gauge, explainability waterfall, and disclosures.
- `/portfolio`, `/reports`, and `/admin` — authenticated product and controls.

Use a desktop viewport plus a mobile viewport, prefer seeded demo data, and make sure the simulator/privacy notice remains visible whenever the image suggests a decision. Do not include customer data, tokens, or private report downloads in screenshots or GIFs.
